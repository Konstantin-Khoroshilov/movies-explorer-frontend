# movies-explorer-frontend

## Сервис, в котором можно найти фильмы по запросу и сохранить в личном кабинете.

Пользователь вводит в строку поиска ключевые слова и нажимает кнопку «Искать». После этого сайт выполняет два действия:
- отправляет запрос к стороннему сервису с данными о фильмах, получает данные и сохраняет;
- согласно выбранному жанру находит все подходящие фильмы и отображает карточки с ними;
- когда пользователь сохраняет фильм, он отображается в специальном разделе сайта.

Сайт состоит из нескольких страниц:
- Главная. Содержит информацию о выполненном проекте.
- Страница с фильмами. На ней есть форма поиска фильмов и блок с результатами поиска.
- Страница с сохранёнными фильмами. Показывает фильмы, сохранённые пользователем.
- Страница регистрации. Позволяет пользователю зарегистрировать аккаунт.
- Страница авторизации. На ней пользователь может войти в систему.
- Страница редактирования профиля. Пользователь может изменить данные своего аккаунта.

Фронтенд:
- Делает асинхронные запросы к нашему и стороннему API по принципам REST.
- Защищает роуты /saved-movies, /profile и /movies авторизацией.
- Хранит данные о пользователе в глобальной стейт переменной currentUser, созданной с помощью createContext.
- Хранит JWT-токен в localStorage.
- Обрабатывает ошибки от API и показывает сообщение пользователю.
- Показывает прелоадер, пока от сервера идет ответ.
- Кликом по карточке переводит на ютуб-трейлер фильма.
- Валидирует данные в формах: поиска, регистрации, авторизации.
- Позволяет поставить фильму лайк и сохранить его.  
- Отображает разные состояния шапки сайта для авторизованных и неавторизованных пользователей.
- Фильтрует фильмы в соответствии с запросом пользователя.

Использованные технологии:
- HTML 5;
- CSS 3;
- React;
- Git;
- REST API;
- localStorage браузера.
